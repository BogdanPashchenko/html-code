Программы для работы по Scrum

Организовать рабочий процесс большой команды бывает не просто. Задачи копятся, сроки горят и много времени уходит на обсуждение проекта. Здесь и начинаются поиски альтернативных решений проблемы. Найти выход из сложившейся ситуации помогут облачные сервисы, показывающие текущее состояние работы команды.

Инструментов сейчас огромное множество. Как же выбрать тот, который действительно сэкономит время и автоматизирует процесс? В этой статье мы подобрали 3 бесплатные программы с удобным интерфейсом.

SCRUMBLR

Интересный и интуитивно понятный сервис Scrumblr, подходит для одновременной работы в команде. Несколько пользователей могут в реальном времени вносить изменения и отслеживать чужие правки. Есть возможность добавлять колонки и перемещать по ним стикеры, на которых редактируется и выделяется цветом текст.

Плюсы

  • Понятный интерфейс.
  • Бесплатный.
  • Не требуется регистрация.

Минусы

  • Англоязычный сервис.

Asana

Еще один англоязычный сервис, позволяющий коллективно управлять задачами. На рабочем столе Asana располагаются проекты, задачи и подзадачи. У проекта есть менеджер, который отвечает за рабочий процесс. В рамках одного проекта задачи перемещаются из одной секции в другую и могут дробиться на подразделы. На каждом этапе устанавливается дедлайн и ответственные лица. Можно ставить теги и лайки, оставлять комментарии. Когда выполненных задач становится слишком много и на доске не остаётся места, они перемещаются в архив.

К одному рабочему пространству бесплатно могут подключиться до 30 пользователей. Их можно добавлять как ко всему рабочему столу так и выдавать доступ к конкретному проекту. В Asana есть инструмент Worksection, позволяющий разделить задачи по рабочим областям. Этот элемент отлично подойдёт для больших команд с разноплановыми задачами. В Asana отображается информация по текущим задачам во всех проектах

Плюсы

  • Интуитивно понятный интерфейс.
  • В одном рабочем пространстве можно увидеть прогресс по всем проектам.
  • Синхронизация с устройствами и облачное хранилище

Минусы

  1. Нельзя отследить прогресс в процентах.
  2. Нет русскоязычной версии.
  3. Нет встроенного мессенджера и возможности обмена файлами.

Trello

Самая популярная система управления проектами в режиме онлайн. Подойдёт для небольших проектов. Trello легко внедряется в рабочий процесс без долгой адаптации сотрудников. Рабочее пространство выполнено в виде системы досок, списков и карточек, которые распределяются по типам. Каждую из досок можно выделять под отдельные процессы. Например, если команда очень большая, то удобнее будет сделать отдельные доски для отделов.

В базовой версии добавляется неограниченное количество досок, списков, карточек и пользователей. Ограничения функционала накладывается только на объём загружаемых файлов (до 10 МБ) и количество интеграций. А фон досок выбирается из коллекции фотостока Unsplash. Программа универсальна и позволяет сразу оценить прогресс по всем основным процессам. Trello- личный органайзер, дневник, список и как форум.

Для каждого процесса своя доска с ответственными людьми

Плюсы

  • Простой интерфейс.
  • Русскоязычная версия.
  • Бесплатная.
  • Многофункциональная.
  • Наглядная.
  • Интегрируется с другими популярными инструментами.
  • Десктопная и мобильная версия.

Минусы

  1. Не подходит для сложных проектов, если появляется много досок, то в них тяжело ориентироваться.
  2. Требуется время на настройку и интеграцию сторонних сервисов.
  3. Нет диаграммы Ганта, иллюстрирующей график работ по проекту.
